# Cara menjalankan project saya

## **Jalankan program dengan venv**
```
python3 -m venv env
```

## **active kan venv terlebih dahulu di terminal disini saya menggunakan os cacyos pak**
```
source env/bin/activate
```
> **Catatan:** Jangan lupa untuk mengaktifkan `venv` sebelum menginstal library!

---

## **kemudian jalankan untuk install library yang di butuhkan**
```
pip3 install -r requirements.txt
```
